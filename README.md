# File Operations Utility: Backup & Compression

A lightweight demonstration project focused on core file system operations, including automated backups, data restoration, and file compression/decompression.

## 🚀 Features
 **Backup & Restore:** Mechanisms to secure files and revert to previous states.
- **Compression:** Efficient `ZIP` file creation to reduce storage footprint.
- **Decompression:** Reliable `UNZIP` functionality to extract archives.
- **File Management:** Streamlined handling of local directory structures.
